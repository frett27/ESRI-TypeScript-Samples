Table of Contents (TOC) widget for ArcGIS Server JavaScript API

A TOC (Table of Contents) widget for ESRI ArcGIS Server JavaScript API. The namespace is agsjs

For a description and examples of how to use this library, check out the how-to.

class TOC

This class is a Table Of Content widget.

Constructor

Constructor Description
TOC(opts:ags.TOCOptions, srcNodeRef:DOMNode|id) Create a Table Of Contents (TOC)

class TOCLayerInfo

This is an object literal that specify the options for each map rootLayer layer.

Properties

Properties Type Description
collapsed Boolean whether to collapsed the rootLayer layer at beginning. default is false, which means expand if visible, collapse if not.
layer Layer ArcGIS Server layer.
noLegend Boolean whether to skip the legend, and only display layers. default is false.
slider Boolean whether to show slider for each rootLayer to adjust transparency. default is false.
title string title. optional. If not specified, rootLayer name is used.

class TOCOptions

This is an object literal that specify the option to construct a TOC. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
indentSize Number indent size of tree nodes. default to 18.
layerInfos Object[] a subset of layers in the map to show in TOC. each object is a TOCLayerInfo
map esri.Map the map instance. required.
